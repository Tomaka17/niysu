Easy-to-use PHP framework

Niysu is a PHP framework that is designed to be easy to use, but not lack any feature of a real framework.

Easy to use

This example handles URLs like /users/12, reads the corresponding user from the database, and returns the values of the columns as JSON data:

class Foo {
	/**
	 * @url /users/{userID}
	 */
	public function handler($userID, $databaseService, $jsonOutput) {
		$jsonOutput->setData($databaseService->users[['id' => $userID]]);
	}
};

$databaseService and $jsonOutput are services that are automatically created when the route is called.

PHP template engine

If you use a custom template engine, you will have to spend a lot of time learning its syntax and writing missing features. That's why Niysu encourages you to use PHP as a template engine.

The major drawback of PHP is that you have to call htmlspecialchars() whenever you output something inside HTML. This is not needed with Niysu's PHP template service. Anything that is inside PHP tags <?php ?> will be automatically escaped.

	// the template:
	<div>
		<p>Value: <?php echo 'Unescaped < st&ring'; ?></p>
	</div>

	// the route handler:
	public function handler($phpTemplateOutput) {
		$phpTemplateOutput->setTemplate('/path/to/template.php');
	}

	// the output:
	<div>
		<p>Value: Unescaped &lt; st&amp;ring</p>
	</div>
