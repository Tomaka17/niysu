Installation

To install Niysu, the first to do is to install composer. Composer is a dependency manager which allows you to easily install and upgrade any PHP library you depend upon.

Once it is installed, write a composer.json file in the main directory of your website:

{
    "require": {
    	"tomaka17/niysu": "dev-master"
    }
}

This will tell Composer that you have a dependency on Niysu.

Now open a shell, and type:

composer install

This will install Niysu and all its dependencies in the vendor directory.

Now to use Niysu in a PHP script, simply add this line at the top of the script:

>
require 'vendor/autoload.php';			// change the path if you are not in the main directory
